### 🛠 What I'm building

**Backend framework**
- [**typescript-backend-toolkit**](https://github.com/muneebhashone/typescript-backend-toolkit) ⭐ 322 — production-ready Express/TypeScript framework: auto-generated OpenAPI, Artisan-style CLI, plugin system, JWT/sessions, BullMQ, React Email, Socket.io, Django-style admin, Zod validation.
- [**ts-easy**](https://github.com/muneebhashone/ts-easy) ⭐ 45 — minimalist TypeScript starter for Node projects. No fluff, just the essentials.

**AI-agent tooling** (CLIs built for AI agents to research the real web)
- [**gh-research**](https://github.com/muneebhashone/gh-research) — deterministic CLI for agents to research GitHub Issues & Discussions, ranked and aggregated as JSON.
- [**x-search-tool**](https://github.com/muneebhashone/x-search-tool) — forces LLM-native search (Grok x_search, Gemini google_search) for agents.
- [**reddit-search**](https://github.com/muneebhashone/reddit-search) & [**hn**](https://github.com/muneebhashone/hn) — agent-first Reddit and Hacker News research CLIs.

**Developer utilities**
- [**mdpdf-cli**](https://github.com/muneebhashone/mdpdf-cli) — turn a Markdown file or folder into one beautifully styled PDF.
- [**ssh-easy**](https://github.com/muneebhashone/ssh-easy) · [**s3-storage-cli**](https://github.com/muneebhashone/s3-storage-cli) — small CLIs that remove daily friction.
